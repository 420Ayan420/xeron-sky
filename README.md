# building
- run `python -m http.server` (trust me it works, no idea why lmao)
- check at `http://localhost:8000/` in browser

# rendering types
- spherical rendering (3D)
- stereographic
- mercator
- hammer-aitoff

# known bugs
- lines wrap around the edge of the in hammer-aitoff projection
